Deploying Active Directory in a Windows Home Lab

In this project, I built a full Windows Active Directory environment from scratch to simulate an enterprise IT setup. It includes a Windows Server 2022 domain controller, a Windows 10 client, and full DNS, GPO, and scripting integration — all in a virtualized lab.

  • Tools Used: VirtualBox, Windows Server 2022, Windows 10, PowerShell, draw.io
  • Domain: madden.local
  • Server IP: 192.168.1.10 — Client IP: 192.168.1.11
  • Networking: Internal NAT, DNS pointing to DC01

Key Features

  • Promoted Server 2022 to Domain Controller
  • Created custom Organizational Units (IT, HR, Sales)
  • Used PowerShell to create users and set passwords
  • Joined Windows 10 client to the domain
  • Enforced Group Policies: custom wallpaper, blocked control panel, mapped network drives, login banner

Sample PowerShell Snippet

New-ADUser -Name "jmadden" -GivenName "Josh" -Surname "Madden" -SamAccountName "jmadden" -UserPrincipalName "jmadden@madden.local" -AccountPassword (ConvertTo-SecureString "P@ssw0rd123" -AsPlainText -Force) -Enabled $true -Path "OU=IT,DC=madden,DC=local"
