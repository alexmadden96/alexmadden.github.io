🧠 Project: Active Directory Domain Controller Setup (Home Lab)

This project simulates a real-world Windows Server environment where I configured a Domain Controller using Windows Server 2022 and began building an enterprise-like Active Directory setup from scratch.

🛠️ Step 1: Server 2022 VM Prep

  • Created a Windows Server 2022 VM in VirtualBox
  • Fully updated the OS using Windows Update
  • Renamed the server to AD-LAB
  • Set a static IP address:
    • IP: 192.168.68.50
    • Subnet: 255.255.255.0
    • Gateway: 192.168.68.1
    • DNS: 127.0.0.1

⚙️ Step 2: Install Active Directory Domain Services (AD DS)

  • Installed AD DS via Server Manager → Add roles and features
  • Promoted the server to a Domain Controller
  • Created new forest: madden.local
  • Set DSRM password and documented it securely
  • Restarted the server after promotion

🌐 Step 3: DNS Configuration

  • Confirmed DNS role was installed with AD DS
  • Opened DNS Manager → Forwarders
  • Configured external DNS forwarders:
    • 8.8.8.8 (Google)
    • 1.1.1.1 (Cloudflare)
  • Verified internet access using ping/nslookup

👥 Step 4: Create OUs and User Accounts

  • Used Active Directory Users and Computers (ADUC)
  • Created Organizational Units:
    • IT
    • HR
    • Sales
  • Created users and assigned to OUs:
    • ajones (IT)
    • hwatts (HR)
    • psmith (Sales)
  • Set secure passwords, changed password age to 90 days, and added descriptions

📸 Screenshots and video walkthroughs were captured throughout the process to visually document the setup and configuration.
